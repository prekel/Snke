Snke 1.0
Файл справки

Управление:
Клавиши вверх, вправо, вниз, влево изменяют направление движения
Если клавиша зажата по направлению движения, включается режим ускорения
/ - режим удлинения

Окно программы:
1. Длина
2. Координаты головы
3. Направление
4. Координаты еды
5. Расстояние до еды
6. Координаты хвоста
7. Шаги
8. Режим ускорения
9. Время
10. Название карты
11. Рекордная длина

Файлы:
readme.txt
	Файл справки
config.txt
	1. Начальная абцисса головы
	2. Начальная ордината головы
	3. Начальная длина
	4. Начальное направление
	5. Символ головы
	6. Символ еды
	7. Название файла карты
highscore.txt
	Рекордная длина
default.map

	21 строк, 78 столбцов
